# Tablo Crawler

This is the tablo-crawler application in the "Project Shisha" monorepo.

## Getting Started

```bash
# Development
bun run dev

# Build
bun run build

# Test
bun run test
```

## Deployment

This app uses the following deployment strategy:

- Dockerfile: `Dockerfile` (safe strategy)
- Deploy vars: `deploy.vars.yml`
